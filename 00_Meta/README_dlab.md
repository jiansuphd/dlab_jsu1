# Repository Overview & Legacy Deployment Instructions

This repository hosts a collection of web-based interactive applications and simulators, likely built for educational purposes.

## Legacy README: GitHub Pages Deployment

*This is the original instructional text provided with the UTK Digital Learning Lab repository, updated to reflect our current domain.*

---

To generate a new page, place a folder containing an `index.html` file into the repository. 

After pushing to GitHub, the Pages site will refresh (which might take around 15 minutes). Once refreshed, that new page will be live at:

`https://jiansuphd.github.io/dlab_jsu1/<your_folder_name_here>/`

*(Note: Since the repository has been restructured, folders placed inside `10_Projects/` will follow the path `https://jiansuphd.github.io/dlab_jsu1/10_Projects/<your_folder_name_here>/`)*
